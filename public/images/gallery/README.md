# Gallery

Each event album is a folder here. The folder name must match the `slug`
in the event's markdown file at `/src/content/events/<slug>.md`.

## Current albums

- `vrr-memorial-2025/` — VRR Memorial Football Match 2025

## Adding a new event

1. Create a new folder here, e.g. `sports-day-2026/`.
2. Drop photos named `01.jpg`, `02.jpg`, `03.jpg`, etc. (any count, any
   orientation, but 1600px wide is a good target).
3. Drop a `cover.jpg` for the album thumbnail (landscape, ~1600x1000px).
4. Create `/src/content/events/sports-day-2026.md` with the frontmatter
   described in `../README.md`.
5. Commit and push. The site regenerates automatically.
